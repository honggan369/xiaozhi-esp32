# LCS3 Otto Robot (立创实战派 S3 + PCA9685)

该板型将 `otto-robot` 的动作控制能力迁移到立创实战派 S3（`lichuang-dev` 硬件基线），并把 6 路舵机改为 `PCA9685` 驱动。

## 1. 硬件组成

- 立创实战派 S3 开发板
- PCA9685 16 路舵机驱动板
- 6 路舵机
- 外部舵机电源（推荐 5V，电流 >= 3A）

## 2. 舵机通道映射

本板型固定使用 `PCA9685` 通道 0~5：

| Otto 部位 | PCA9685 通道 |
|---|---|
| Right Leg | CH0 |
| Right Foot | CH1 |
| Left Leg | CH2 |
| Left Foot | CH3 |
| Left Hand | CH4 |
| Right Hand | CH5 |

## 3. 连线示意（实战派 S3 -> PCA9685）

实战派 S3 的 I2C 与音频/触摸共总线，PCA9685 挂到同一组 I2C 线上：

| 实战派 S3 | PCA9685 | 说明 |
|---|---|---|
| GPIO1 (SDA) | SDA | I2C 数据 |
| GPIO2 (SCL) | SCL | I2C 时钟 |
| GND | GND | 必须共地 |
| 3V3 | VCC | PCA9685 逻辑电源 |

舵机接线（每路三线）：

- 舵机信号线 -> PCA9685 `CHx` 对应信号针脚
- 舵机电源正极 -> PCA9685 `V+`（外部 5V 电源正极）
- 舵机地线 -> PCA9685 GND（与开发板 GND 共地）

## 4. 供电建议（重要）

建议采用“双路供电、单地共地”：

1. 立创实战派 S3：USB-C 供电（或稳定 5V 给开发板 5V 输入）。
2. PCA9685 舵机电源：独立外部 5V 大电流电源接 `V+`。
3. PCA9685 `VCC` 只接 3V3 逻辑电源，不要接到 5V。
4. 开发板 GND、PCA9685 GND、外部舵机电源 GND 必须共地。

不建议用开发板直接给 6 路舵机供电，否则容易压降、重启、噪声串扰。

## 5. 功能说明

- 保留 `otto-robot` 动作控制工具链（`self.otto.action` 等）。
- 保留 WebSocket 控制页面（端口 `8080`）。
- 保留立创实战派原有音频、显示、触摸、摄像头能力。
- 舵机输出由板载 GPIO PWM 改为 `PCA9685 PWM`，避免占用高风险引脚。
